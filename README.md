#### Executando o tutorial localmente

1.  Clone o repositório

2. Instale as dependências  

```bash
curl -s https://deb.nodesource.com/setup_16.x | sudo bash
sudo apt install nodejs

node --version
npm --version

npm run docs:dev
```
